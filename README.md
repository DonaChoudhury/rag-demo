# 🧮 Demo Calculator Repo for RAG Project

Welcome to the demo repository! This codebase is explicitly created to test the data ingestion pipeline of an AI-powered Chat-with-Codebase application.

## 🚀 Features
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division).
- Safe division handling to prevent `ZeroDivisionError` crashes.

## 🛠️ How to use
Import the `calculator.py` module into your main application and call the functions directly.

```python
import calculator

result = calculator.add(10, 5)
print(result) # Output: 15
